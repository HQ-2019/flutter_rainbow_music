# flutter_rainbow_music

#### 简介

彩虹音乐是一款音乐播放器，愿动听的音乐能让我们天天好心情。
彩虹音乐是一个我闲暇时间自娱自乐的项目，有时间就写一些。项目会使用到酷狗的音乐API。如有侵权，请联系我删除。
